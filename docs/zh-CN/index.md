## 为什么用它

`vue3-monaco-editor` 适合需要一个轻量、类型完善的 SQL 编辑器封装的项目：

- 通过 `v-model` 绑定编辑器内容
- 通过 Vue props 切换 Monaco 主题
- 通过宽高 props 控制编辑器尺寸
- 在需要时传入 Monaco 原生配置
- 用数据库元数据驱动 SQL 补全
