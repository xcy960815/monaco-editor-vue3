# vue3-monaco-editor

一个轻量的 Vue 3 Monaco Editor 组件库。

## 特性

- 支持 Vue 3 组件和插件安装。
- 支持编辑器内容 `v-model` 双向绑定。
- 支持 SQL 关键字、库、表、字段补全。
- 支持主题、宽度、高度和 Monaco 原生配置。

## 快速开始

```shell
pnpm add vue3-monaco-editor monaco-editor
```

```ts
import MonacoEditor from 'vue3-monaco-editor'
import 'vue3-monaco-editor/style.css'
```
